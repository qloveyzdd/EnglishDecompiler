# English Decompiler

对外主入口请以 [README.md](./README.md) 为准。

[查看英文主 README](./README.md)

English Decompiler 是一个面向英文技术文档的白盒解析器。
它不是通用翻译器，也不是英语学习工具。

它想做的不是直接给你一个“翻译答案”，而是把技术英文拆成开发者可以检查的结构，比如 `action`、`object`、`relation`、`condition` 和 `purpose`。

核心一句话：

> Translation gives you an answer. Parsing gives you control.
