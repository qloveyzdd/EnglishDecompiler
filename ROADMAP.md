# English Decompiler Roadmap

This is the public roadmap for turning English Decompiler from a repository skeleton into a usable parser, demo, IR surface, and launch-ready open-source project.

## Phase 1: Repository Positioning and Launch Skeleton

Goal: Make the repository understandable at a glance and give visitors a credible public entry point.

## Phase 2: Parser Core v0

Goal: Parse single technical-English sentences into stable roles and structured JSON.

## Phase 3: Local Demo v0

Goal: Expose the parser through a minimal local demo that shows source text, token roles, and structured output side by side.

## Phase 4: IR and White-Box Correction Loop

Goal: Turn parser output into an editable reasoning surface instead of static output.

## Phase 5: Launch Assets and Feedback Intake

Goal: Package the MVP for public launch and collect real-world parser failures from developers.

## Repository launch metadata

- About: A white-box parser for English technical docs.
- Supporting tagline: Parse technical English like code.
- Topics: developer-tools, nlp, english-parser, technical-documentation, translation, llm, language-tools, documentation-tools
- Social preview direction: show the project name, the supporting tagline, and a sentence -> structure -> IR visual path
